# Required Skills

- **Frontend Development**: HTML5, CSS3, TypeScript (Vanilla)
- **DOM Manipulation**: Native API (`document.createElement`, `appendChild`)
- **UI/UX Design**: Responsive design, Complex layouts (Flexbox/Grid)
- **State Management**: Custom observable/store pattern or simple global state
- **Tooling**: Vite
- **Design System Implementation**: CSS Variables, Theming
